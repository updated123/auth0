Available MFA Methods
Most Secure
TOTP — Authenticator App

Google Authenticator, Authy, 1Password, or any TOTP-compatible app. Generates a 6-digit code every 30 seconds.

Works offline No SIM swap Industry standard Requires app install
How it works: User scans a QR code once. App generates TOTP codes locally — no network required.
Easy for Users
SMS OTP

A one-time code sent via text message to the user's phone number. Supported by Auth0 natively.

No app needed Familiar to all users SIM swap risk Needs phone signal
How it works: Auth0 sends a 6-digit code via SMS. User enters it within 5 minutes. Requires Twilio integration.
Zero Friction
Email OTP

A one-time code sent to the user's verified email address. Zero infrastructure — works with existing email setup.

Zero setup Works everywhere Email inbox access = bypass Lower security than TOTP
How it works: Auth0 emails a 6-digit code to the user's verified address. No additional configuration needed.
